Web Stocket

Es una tecnologia que proporciona un canalde de comunicacion bidireccional y fullduplex sobre un unico TCP y esta diseñada para navegadores y servidores,pero tambien puede utilizarse en cualquier aplicación que sea tipo cliente servidor.

Local Storage

Lo que se podían llamar Cookie 2.0 es el Local Storage utilizado en HTML5 para almacenar datos con un máximo de 5Mb. De una manera más sencilla y más segura que las cookie normales.
